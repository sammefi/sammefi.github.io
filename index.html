Music Score Project

Intention of this project is to create a library (Music Score Lib) that can be used to show and play musical scores/notation on web pages using JavaScript/TypeScript. Comes with React component but can be used without as well.

Author is hobbyist programmer and classical guitar student. Focus is currently on classical guitar notation.

For more info:

  • Repository coming soon. This is a monorepo containing workspaces for the library and test applications.
  • Documentation for the library.
  • NPM package maybe coming later.
